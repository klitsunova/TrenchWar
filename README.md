# TrenchWar
TrenchWar - некоммерческая 2D-игра в жанре Tower Defense, написанная на C++ с использованием Qt framework (6.3.0+). Игра предназначена для ПК. 

# Статус разработки

Активный этап разработки окончен в июне 2022.
